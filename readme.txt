Menus for your WP-Admin sidebar that will save you time. Powered by jQuery and CSS.

== Description ==

Start saving time now. Make use of this lightweight and simple Pop Menu plugin for the new WP-Admin backend.

Benifits:

*  Saves time.
*  Saves space.
*  Improves usability.
*  Acess to any menus on demand.
*  Unobtrusive code.
*  Created with the jQuery library

Known Bugs:

*  Page must be refreshed if menu is collapsed/folded.
*  IE6 - Menus go under Select Boxes and some hover issues.
*  IE - No fixed menu for Intenet Explorer
*  Still to be tested with Opera


== Installation ==

1. Upload `pop_menus.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start using the Wp-Admin Sidebar and see the pop menus at work.
